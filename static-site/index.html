🛍️ ShopAPI

Cloud-based e-commerce REST API — product catalogue with orders, JWT authentication, full CRUD.

Node.js + ExpressJSON File DBJWT AuthSwagger/OpenAPI14 Unit Tests

Overview

🛒

Products

Full CRUD — list, view, create, update, delete with stock tracking.

📦

Orders

Place orders with multiple items, track status, cancel with stock restore.

🔐

JWT Auth

Register & login. Mutating endpoints require a Bearer token.

📄

Swagger UI

Interactive docs at /api-docs — try every endpoint live.

Endpoints

MethodPathDescriptionAuthResponses
POST/auth/registerRegister new user—201 · 422
POST/auth/loginLogin, get JWT—200 · 401
GET/productsList all products—200
GET/products/:idGet product by ID—200 · 404
POST/productsCreate product🔒201 · 401 · 422
PUT/products/:idUpdate product🔒200 · 401 · 404
DELETE/products/:idDelete product🔒204 · 401 · 404
GET/ordersList my orders🔒200 · 401
GET/orders/:idGet order by ID🔒200 · 403 · 404
POST/ordersPlace an order🔒201 · 422
PUT/orders/:idUpdate order status🔒200 · 422
DELETE/orders/:idCancel order🔒204 · 422

Examples

Register

POST /auth/register
{"username":"john","email":"john@ex.com","password":"secret123"}
→ 201 {"message":"User registered successfully"}

Login

POST /auth/login
{"email":"john@ex.com","password":"secret123"}
→ 200 {"token":"eyJhbGci...","user":{...}}

Get Products

GET /products
→ 200 [{"id":1,"name":"Laptop Pro","price":1299.99,...}]

Place Order

POST /orders  (Bearer token required)
{"items":[{"product_id":1,"quantity":1}]}
→ 201 {"id":1,"status":"pending","total_price":1299.99}

Run Locally

cd ecommerce-api
npm install
node server.js
# API:   http://localhost:3000
# Docs:  http://localhost:3000/api-docs
# Tests: npm test
